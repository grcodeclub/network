Αγγλικά Ελληνικά Περιγραφή
Router Δρομολογητής Συσκευή που κατευθύνει πακέτα δεδομένων μεταξύ διαφορετικών δικτύων.
Switch Διακόπτης Συσκευή που συνδέει συσκευές σε ένα τοπικό δίκτυο (LAN) και ανακατευθύνει τα δεδομένα μεταξύ τους.
Hub Κόμβος Συσκευή που συνδέει πολλές συσκευές σε ένα δίκτυο, διαβιβάζοντας δεδομένα σε όλους τους συνδεδεμένους υπολογιστές.
Modem Μόντεμ Συσκευή που μετατρέπει ψηφιακά δεδομένα σε αναλογικά σήματα και αντίστροφα για τη σύνδεση με το Διαδίκτυο.
Access Point Σημείο Πρόσβασης Συσκευή που επιτρέπει σε ασύρματα συσκευές να συνδέονται σε ένα ενσύρματο δίκτυο.
IP Address Διεύθυνση IP Μοναδικός αριθμός που αναγνωρίζει μια συσκευή σε ένα δίκτυο.
Subnet Υποδίκτυο Μέρος ενός μεγαλύτερου δικτύου που έχει διαχωριστεί για να διευκολύνει τη διαχείριση.
Subnet Mask Μάσκα Υποδικτύου Διευκρινίζει την περιοχή του δικτύου και τη διεύθυνση υποδικτύου ενός IP.
Gateway Πύλη Συσκευή που συνδέει ένα τοπικό δίκτυο με άλλα δίκτυα, συμπεριλαμβανομένων των δημόσιων δικτύων.
Bandwidth Εύρος Ζώνης Η χωρητικότητα του δικτύου για τη μεταφορά δεδομένων, συνήθως μετρημένη σε Mbps ή Gbps.
Packet Πακέτο Μια μονάδα δεδομένων που μεταφέρεται σε ένα δίκτυο.
Protocol Πρωτόκολλο Σύνολο κανόνων που καθορίζουν τον τρόπο με τον οποίο επικοινωνούν τα συστήματα στο δίκτυο.
Packet Switching Αλλαγή Πακέτων Μέθοδος μετάδοσης δεδομένων σε δίκτυα, όπου τα δεδομένα διασπώνται σε πακέτα που στέλνονται ξεχωριστά.
Circuit Switching Αλλαγή Κυκλώματος Μέθοδος μετάδοσης δεδομένων όπου μια φυσική διαδρομή δημιουργείται για τη διάρκεια της επικοινωνίας.
OSI Model Μοντέλο OSI Μοντέλο 7 επιπέδων που περιγράφει τη λειτουργία των δικτύων από το φυσικό επίπεδο έως την εφαρμογή.
TCP/IP Πρωτόκολλο TCP/IP Σύνολο πρωτοκόλλων που χρησιμοποιούνται για τη σύνδεση υπολογιστών στο Διαδίκτυο.
Ethernet Εθερνέτ Σύνολο τεχνολογιών για την τοπική δικτύωση μέσω καλωδίων.
Fiber Optic Οπτικές Ίνες Τεχνολογία μετάδοσης δεδομένων που χρησιμοποιεί ίνες από γυαλί ή πλαστικό για υψηλή ταχύτητα και μικρές απώλειες.
Topology Τοπολογία Η διάταξη των συσκευών και των συνδέσεων σε ένα δίκτυο.
Star Topology Τοπολογία Αστεριού Δίκτυο όπου όλες οι συσκευές συνδέονται σε έναν κεντρικό διακόπτη ή δρομολογητή.
Mesh Topology Τοπολογία Πλέγματος Δίκτυο όπου κάθε συσκευή συνδέεται με όλες τις άλλες, εξασφαλίζοντας υψηλή αξιοπιστία.
Bus Topology Τοπολογία Διαύλου Δίκτυο όπου όλες οι συσκευές συνδέονται σε ένα κοινό καλώδιο διαύλου.
Client-Server Model Μοντέλο Πελάτη-Εξυπηρετητή Αρχιτεκτονική όπου οι πελάτες αιτούνται υπηρεσίες από κεντρικούς εξυπηρετητές.
Peer-to-Peer (P2P) Ομότιμοι (P2P) Μοντέλο δικτύωσης όπου οι συσκευές λειτουργούν και ως πελάτες και ως εξυπηρετητές.
MAC Address Διεύθυνση MAC Μοναδικός αριθμός που αναγνωρίζει κάθε συσκευή δικτύου σε ένα τοπικό δίκτυο.
Port Θύρα Λογική διασύνδεση που χρησιμοποιείται για την κατεύθυνση δεδομένων σε εφαρμογές ή υπηρεσίες.
Virtual Private LAN Service (VPLS) Υπηρεσία Εικονικού Ιδιωτικού Τοπικού Δικτύου (VPLS) Υπηρεσία που επεκτείνει ένα τοπικό δίκτυο μέσω γεωγραφικών αποστάσεων.
Content Delivery Network (CDN) Δίκτυο Παράδοσης Περιεχομένου (CDN) Δίκτυο διακομιστών που παρέχει περιεχόμενο κοντά στον χρήστη για ταχύτερη παράδοση.
Bandwidth Throttling Περιορισμός Εύρους Ζώνης Η πρακτική της μείωσης της διαθέσιμης χωρητικότητας του δικτύου για τον έλεγχο της κυκλοφορίας.
Ping Ping Εργαλείο που ελέγχει την προσβασιμότητα ενός διακομιστή ή συσκευής σε ένα δίκτυο.
Traceroute Traceroute Εργαλείο που καταγράφει τη διαδρομή που ακολουθούν τα πακέτα για να φτάσουν σε έναν προορισμό.
Network Security Ασφάλεια Δικτύου Πρακτικές και τεχνολογίες που προστατεύουν ένα δίκτυο από επιθέσεις και μη εξουσιοδοτημένη πρόσβαση.
IP Routing Δρομολόγηση IP Η διαδικασία καθορισμού της διαδρομής που ακολουθούν τα πακέτα δεδομένων για να φτάσουν στον προορισμό τους.
Network Interface Διεπαφή Δικτύου Η σύνδεση μεταξύ ενός υπολογιστή και ενός δικτύου.
Network Topology Τοπολογία Δικτύου Η διάταξη και η σύνδεση των συσκευών σε ένα δίκτυο.
Network Congestion - Η κατάσταση όταν ένα δίκτυο έχει υπερβολική κυκλοφορία, οδηγώντας σε καθυστερήσεις και απώλειες δεδομένων.
Network Segmentation Τμηματοποίηση Δικτύου Η διαδικασία διαχωρισμού ενός δικτύου σε μικρότερα τμήματα για καλύτερη διαχείριση και ασφάλεια.
Network Performance Απόδοση Δικτύου Η αξιολόγηση της ταχύτητας και της αποτελεσματικότητας ενός δικτύου.
Network Management Διαχείριση Δικτύου Η διαδικασία ελέγχου και συντήρησης της υποδομής ενός δικτύου.
Network Troubleshooting Επίλυση Προβλημάτων Δικτύου Η διαδικασία εντοπισμού και διόρθωσης προβλημάτων σε ένα δίκτυο.
Bandwidth Management Διαχείριση Εύρους Ζώνης Η διαδικασία ελέγχου της χωρητικότητας του δικτύου για την αποφυγή συμφόρησης και βελτίωση της απόδοσης.
Network Architecture Αρχιτεκτονική Δικτύου Η σχεδίαση της δομής και της λειτουργίας ενός δικτύου.
Throughput Ροή Δεδομένων Η ποσότητα των δεδομένων που μεταφέρονται μέσω ενός δικτύου σε μια δεδομένη χρονική περίοδο.
Network Packet Πακέτο Δικτύου Μικρή μονάδα δεδομένων που μεταφέρεται μέσω ενός δικτύου.
Data Rate Ρυθμός Δεδομένων Η ταχύτητα με την οποία τα δεδομένα μεταφέρονται μέσω ενός δικτύου.
Bandwidth Allocation Κατανομή Εύρους Ζώνης Η διαδικασία κατανομής της διαθέσιμης χωρητικότητας του δικτύου σε διάφορους χρήστες ή εφαρμογές.
Network Bandwidth Εύρος Ζώνης Δικτύου Η μέγιστη χωρητικότητα ενός δικτύου για τη μεταφορά δεδομένων.
Firewall Τείχος Προστασίας Συσκευή ή λογισμικό που ελέγχει και περιορίζει την κυκλοφορία δεδομένων εισερχόμενη ή εξερχόμενη από ένα δίκτυο.
Wireless Network Ασύρματο Δίκτυο Δίκτυο που χρησιμοποιεί ραδιοκύματα για την επικοινωνία αντί για καλώδια.
Network Interface Card (NIC) Κάρτα Διεπαφής Δικτύου (NIC) Συσκευή που επιτρέπει σε έναν υπολογιστή να συνδεθεί σε δίκτυο.
Bridge - Συσκευή που συνδέει δύο δίκτυα ή τμήματα δικτύου ώστε να λειτουργούν ως ενιαίο δίκτυο.
Proxy Server Διακομιστής Αντιπροσωπείας Συσκευή ή λογισμικό που λειτουργεί ως μεσάζων μεταξύ πελάτη και εξωτερικών διακομιστών.
Data Center Κέντρο Δεδομένων Εγκατάσταση που φιλοξενεί διακομιστές και άλλα δίκτυα υποδομών για την αποθήκευση και διαχείριση δεδομένων.
IP Addressing Διευθυνσιοδότηση IP Η διαδικασία εκχώρησης και διαχείρισης διευθύνσεων IP σε συσκευές ενός δικτύου.
Cloud ComputingΥπολογιστικά ΝέφηΗ χρήση διαδικτυακών υπηρεσιών για την αποθήκευση, διαχείριση και επεξεργασία δεδομένων αντί για τοπικό υπολογιστικό εξοπλισμό.
Wi-Fi-Μια τεχνολογία που επιτρέπει την ασύρματη σύνδεση υπολογιστών και άλλων συσκευών σε ένα δίκτυο χρησιμοποιώντας ραδιοκύματα.
5G5GΗ πέμπτη γενιά τεχνολογίας κινητής τηλεφωνίας, προσφέροντας ταχύτερες ταχύτητες μεταφοράς δεδομένων και μειωμένη καθυστέρηση.
TelecommunicationsΤηλεπικοινωνίεςΗ τεχνολογία και οι διαδικασίες που χρησιμοποιούνται για την επικοινωνία σε απόσταση μέσω τηλεφώνων, ραδιοφώνου, τηλεόρασης και άλλων μέσων.
Server FarmΦάρμα ΔιακομιστώνΣυλλογή από πολλούς διακομιστές που χρησιμοποιούνται για τη διαχείριση μεγάλου όγκου δεδομένων και την παροχή υπηρεσιών.
