Ακρωνύμια - Περιγραφή

 🔗Περισσότερες Συντομογραφίες

ΑκρωνύμιοΠλήρες ΌνομαΠεριγραφή
QoSQuality of ServiceΜηχανισμός που διαχειρίζεται την απόδοση του δικτύου και την προτεραιότητα των δεδομένων.
WANWide Area NetworkΔίκτυο που καλύπτει ευρύτερες γεωγραφικές περιοχές, όπως πόλεις ή χώρες.
LANLocal Area NetworkΔίκτυο που καλύπτει περιορισμένες γεωγραφικές περιοχές, όπως ένα κτίριο ή μια πανεπιστημιούπολη.
MANMetropolitan Area NetworkΔίκτυο που καλύπτει μια μεσαία γεωγραφική περιοχή, όπως μια πόλη.
VPNVirtual Private NetworkΔίκτυο που παρέχει ασφαλή σύνδεση μέσω του δημόσιου διαδικτύου.
MACMedia Access ControlΔιεύθυνση που χρησιμοποιείται για την αναγνώριση μιας συσκευής σε ένα δίκτυο.
SSIDService Set IdentifierΌνομα που δίνεται σε ένα ασύρματο δίκτυο.
SSLSecure Sockets LayerΠρωτόκολλο κρυπτογράφησης για την ασφαλή επικοινωνία στο διαδίκτυο.
TLSTransport Layer SecurityΣύγχρονη έκδοση του SSL για την ασφαλή επικοινωνία μέσω δικτύου.
CDNContent Delivery Network (CDN) Δίκτυο διακομιστών που παρέχει περιεχόμενο κοντά στον χρήστη για ταχύτερη παράδοση.
VLANVirtual Local Area NetworkΔίκτυο που διαχωρίζει φυσικά δίκτυα σε εικονικά τμήματα για καλύτερη διαχείριση.
WLANWireless LANΑσύρματο Τοπικό Δίκτυο
SSIDService Set IdentifierΌνομα που δίνεται σε ένα ασύρματο δίκτυο για να αναγνωριστεί από τις συσκευές.
URLUniform Resource LocatorΔιεύθυνση που υποδεικνύει την τοποθεσία ενός πόρου στο διαδίκτυο
UDPUser Datagram ProtocolΠρωτόκολλο μεταφοράς δεδομένων που είναι γρήγορο αλλά μη αξιόπιστο, χωρίς επιβεβαίωση παράδοσης
TCPTransmission Control ProtocolΠρωτόκολλο μεταφοράς δεδομένων που παρέχει αξιόπιστη σύνδεση και επιβεβαίωση παράδοσης
ICANNInternet Corporation for Assigned Names and NumbersΟργανισμός που διαχειρίζεται τις διευθύνσεις IP και τα ονόματα τομέα στο διαδίκτυο.
RDPRemote Desktop ProtocolΠρωτόκολλο που επιτρέπει την απομακρυσμένη σύνδεση και έλεγχο υπολογιστών μέσω δικτύου.
PANPersonal Area NetworkΔίκτυο που καλύπτει μικρή απόσταση, συνήθως γύρω από ένα άτομο, για τη σύνδεση προσωπικών συσκευών.
WPAWi-Fi Protected AccessΠρωτόκολλο ασφαλείας για την προστασία των ασύρματων δικτύων Wi-Fi.
WPA2Wi-Fi Protected Access 2Βελτιωμένη έκδοση του πρωτοκόλλου WPA με ισχυρότερα μέτρα ασφαλείας.
GSMGlobal System for Mobile CommunicationsΠρότυπο για την κινητή τηλεφωνία που χρησιμοποιείται παγκοσμίως για τη μετάδοση φωνής και δεδομένων.
