ARP

Δυναμικός μηχανισμός εντοπισμού της MAC Διεύθυνσης μιας συσκευής στο δίκτυο, χρησιμοποιούντα την αντίστοιχη IP address της συσκευής.

Για να εντοπιστεί η MAC μιας συσκευής χρησιμοποιεί

  1. ARP Reuest που είναι Broadcast και κάνουμε ερώτηση προς το δίκτυο
  2. ARP Reply που είναι Unicast και είναι η απάντηση με την MAC Διεύθυνση

Στον Πίνακα ARP αποθηκεύουμε τα ζεύγη IP και MAC για τις συσκευές του δικτιού

ARP Request

  1. Κωδικός Λειτουργίας : 1
  2. Το πεδίο MAC Address του παραλήπτη έχει όλα τα bit 0
  3. Το πεδίο MAC Address του είναι γνωστό
  4. Τα πεδία IP Address του είναι γνωστά

Το ARP Request ενθυλακώνεται σε ένα Ethernet Frame οπού :

  1. Η MAC αποστολέα είναι γνωστή
  2. Η MAC παραλήπτη είναι Broadcast, γιατί απευθύνεται σε όλους του Η/Υ του δίκτυο

ARP Reply

Η συσκευή που θα βρει ότι η IP αποστολέα στο arp reguest είναι η δική της θα απάντηση με ένα ARP Reply.
Το ARP Reply είναι UNICAST αφού γνωρίζει την IP και MAC διεύθυνση του παραλήπτη (από το ARP Reguest)
Στο ARP πακέτο ο κωδικός λειτουργίας θα είναι 2
