Πρωτόκολλα

Τα πρωτοκόλλα είναι ένα σύνολο από κανόνες που καθορίζουν τον τρόπο θα μπορούν να επικοινωνούν και να ανταλλάζουν πληροφορίες δυο συσκευές μεταξύ τους. Ορισμένα τα οποία ορίζει ένα πρωτόκολλο είναι το είδος της πληροφορίας, την μορφή και την σειρά των μνημάτων που θα ανταλλάξουν οι δυο συσκευές.


Αναζήτηση
Όρος Ελλήνικα Περιγραφή
Network Δίκτυο Ένα δίκτυο αναφέρεται σε ένα σύνολο συνδεδεμένων συσκευών που μπορούν να επικοινωνούν μεταξύ τους.
Node Κόμβος Ένας κόμβος αναφέρεται σε μια συσκευή ή ένα σημείο στο δίκτυο, όπως υπολογιστής ή router.
Router Δρομολογητής Ένα Router είναι μια συσκευή που συνδέει διάφορα δίκτυα μεταξύ τους και μπορεί να αποφασίζει τη διαδρομή που πρέπει να ακολουθήσουν τα IP πακέτα.
Protocol Πρωτόκολλο Ένα πρωτόκολλο αναφέρεται σε ένα σύνολο κανόνων και διαδικασιών που χρησιμοποιούνται για την επικοινωνία μεταξύ δικτύων και συσκευών.
Internet Protocol (IP) Το IP είναι ένα πρωτόκολλο διαδικτύου που χρησιμοποιείται για τη μεταφορά δεδομένων μεταξύ συσκευών σε ένα δίκτυο. Το IPv4 και το IPv6 είναι οι δύο κύριες εκδόσεις του πρωτοκόλλου IP.
Media Access Control (MAC) Η διεύθυνση MAC αναφέρεται είναι ένα μοναδικό αναγνωριστικό που ανατίθεται σε κάθε δικτυακή κάρτα. Χρησιμοποιείται για την αναγνώριση μιας συσκευής σε ένα LAN.
Network Address Translation (NAT) Είναι η τεχνική που χρησιμοποιείται για να μετατρέψει μια διεύθυνση IP σε άλλη διεύθυνση IP κατά τη διέλευση από ένα δίκτυο σε ένα άλλο.
Access Point Συσκευή ή σημείο πρόσβασης ασύρματης σύνδεσης.
Bandwidth Εύρος ζώνης συχνοτήτων. Επίσης χρησιμοποείται για να δείξει τo εύρος των δεδομένων που μπορούν να μεταδοθούν σε ένα τηλεπικοινωνιακό κανάλι.
Broadcast Εκπομπή προς όλους τους κόμβους του δικτύου, συνοδεύοντας την πληροφορία με την αντίστοιχη broadcast διεύθυνση, ώστε να αποσταλεί σε όλους τους κόμβους.
Buffer Χώρος προσωρινής αποθήκευσης
Forwarding Προώθηση πακέτων Μετακίνηση ενός πακέτου από την θύρα που εισέρχεται προς την κατάλληλη θύρα του router.
Local Area Network (LAN) Ένα δίκτυο συσκευών που καλύπτει μια μικρή γεωγραφική περιοχή, όπως τα γραφεία μιας εταιρείας ή τον χώρο ενός σχολείου ή μιας πανεπιστημιακής σχολής κλπ
